# Espacio Ludimente

Prototipo de sitio web para servicios de psicología escolar y terapia de juego.

## Propósito
Este proyecto es el entregable final del módulo de Intro a Desarrollo Web. El tema es crear un prototipo de una landing page de un centro que ofrezca información sobre servicios psicológicos para padres y escuelas.

## Tecnologías
- HTML5
- CSS3
- Git & GitHub
